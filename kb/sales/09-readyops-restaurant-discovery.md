# ReadyOps Restaurant Discovery

**Audience:** Sales technicians
**Read time:** 8 minutes
**Use this when:** Qualifying restaurants, cafes, bars, fast-casual operators, or food-service groups.

> **Sales outcome:** Build a complete picture of the restaurant’s connected operations and determine whether the Restaurant Operations Add-on fits.

## Start with business flow

Ask what happens from order to fulfillment:

```text
Customer order → POS/payment → printer/KDS → kitchen → pickup/delivery → reporting
```

Then identify every network or vendor dependency.

## Required inventory

| Area | Questions |
|---|---|
| POS | Vendor, terminal count, payment-terminal count, upcoming change |
| Kitchen | Receipt/kitchen printers, KDS screens, network dependencies |
| Internet | Primary carrier, backup connection, outage history |
| Wi-Fi | AP count, staff/guest networks, dead zones, handheld POS |
| Delivery | DoorDash, Uber Eats, Grubhub, POS injection, tablets |
| Customer tools | QR menu, reviews, loyalty, gift cards, birthday/VIP |
| Monitoring | Devices that should generate an outage alert |

## POS readiness

Recommend readiness assessment before a new POS or cutover when there are unknowns involving cabling, AP coverage, printers, payment terminals, VLANs, or vendor requirements.

Never claim to replace the POS vendor. ReadyTech owns network readiness, connectivity, coordination, testing, and documented handoff.

## Restaurant Operations Add-on

Best fit when the customer needs ongoing support for:

- POS network monitoring
- Supported printers/KDS
- Delivery-app coordination
- Guest Wi-Fi
- QR/loyalty tools
- Restaurant-vendor coordination

Typical starting combination:

```text
ReadyContinuity Pro                 $449/month
Restaurant Operations Add-on       $250/month
Typical starting total             $699/month/location
```

## High-risk triggers

- Overnight cutover
- Multiple locations
- Unsupported legacy POS
- No network documentation
- Shared guest/POS network
- Frequent printer/KDS failures
- No tested backup internet
- Cabling/construction work

These require the Uptime Assessment and may require engineering review.

## Simple talk track

> “We are not just installing Wi-Fi. We are mapping the restaurant systems that depend on connectivity, separating critical traffic, monitoring the right devices, and coordinating the vendors so the owner has one technical point of contact.”
